### Siguiente paso: Instalación del SO

Como ya tienes la documentación lista, el siguiente paso es la **acción**.
Procede con la instalación de **Ubuntu Server 24.04** en el ThinkPad.

**Recordatorio rápido de la configuración crítica durante la instalación:**
1.  **Disco:** Usa solo el SSD de 120GB para `/`. Deja el de 500GB tranquilo.
2.  **SSH:** Marca [x] Install OpenSSH Server.
3.  **Usuario:** Crea tu usuario (ej: `admin-lab`).

Cuando termine la instalación y veas el `login:` en la pantalla del portátil, avísame. Ahí empezaremos la **Fase 2** (Conectarnos desde tu PC Windows y configurar las llaves SSH).
